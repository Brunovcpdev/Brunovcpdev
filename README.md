### 📌 Projetos em Destaque

**[Cadastro de Clientes](https://github.com/Brunovcpdev/CadastroClientes)**
`C#` `ASP.NET Core` `SQL Server` `HTML` `Bootstrap`
Sistema completo de gerenciamento de clientes com API REST integrada ao SQL Server via Stored Procedures. Endpoints para criar, listar, editar e excluir clientes, com validação de campos e interface responsiva.

**[Tarefas Domésticas](https://github.com/Brunovcpdev/tarefas-domesticas)**
`C#` `.NET` `SQL Server`
Aplicação para cadastro e gerenciamento de tarefas com persistência de dados em SQL Server. Permite criar, listar e organizar tarefas com controle de status.

**[Tarefas Domésticas Cleaner](https://github.com/Brunovcpdev/tarefas-domesticas-cleaner)**
`C#` `Console App` `SQL Server`
Console Application para gerenciamento e limpeza automática de tarefas vencidas, aplicando lógica de negócio para identificar e remover registros expirados no SQL Server.
